Ghost Makers - HModa - Documentation

The documentation of Ghost Makers - HModa

Quick Start →

Elegant Web App with Vue.js

The website is made with Vue.js 2, one of the most popular front-end frameworks.

Solid Rest APIs with Node.js

The APIs are made with Node.js and Express.js, they use cookies as a wrapper for JWT in order to provide both performance and security. Every request is validated by keeping security in mind.

Clear code with Typescript

Both the Web App and the APIs are written in Typescript, so that bugs can be discovered during development.

Fast database with MongoDB

MongoDB is used as database to provide performance and flexibility.

Easy translations with POEditor

The frontend uses i18n and json files for the translations, that are mapped on POEditor so that anyone can update them.

Even an SDK for the APIs

The Web Ap uses an SDK, published to NPM, to make the calls to the APIs. This way additional services written in javascript would have an easy way to interface with the APIs.
